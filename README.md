# Open Linux Workshop
In diesem Workshop wird der Einsatz und das Arbeiten von Linux in einem Enterprise-Umfeld erarbeitet. Nutzer ohne Erfahrung mit dieser Technologie sollen am Schluss in der Lage sein Linux zu administrieren und zu verwenden.

## Konzept
Der Kurs nutzt das Inverted Classroomverfahren. Es wird ein gemeinsamer Kickoff-Termin veranstaltet. Anschließend erhalten die Teilnehmer Hausübungen in Form von Challenges, die alleine oder im Team zu bewältigen sind.

Während der Challengephase gibt es einen FAQ Termin. Dort können Fragen gestellt und Showstopper beseitigt werden.

Am Schluss der Challengephase wird ein Merge-Termin abgehalten. Die Erkenntnisse der Teilnehmer werden zusammengetragen und zu einem Ergebnis verarbeitet. Das Ergebnis umfasst die Linuxlandschaft im Betrieb und die Art und Weise wie Linux in Zukunft innerhalb der Organisation eingesetzt werden soll.

Dieser Workshop ist agnostisch gegenüber der gewählten Distro.
