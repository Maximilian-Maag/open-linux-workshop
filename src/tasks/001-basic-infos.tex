    \section{Intro}
    Linux ist eine Alternative zu Windows und MacOS. Es ist aber kein Betriebssystem sondern ein Kernl.
    Erst mit den tools des GNU Projekts ist Linux ein vollständiges OS. \\\\
    Bevor wir uns auf Linux stürzen stellt sich die Frage: Was ist ein Betriebssystem? \\\\
